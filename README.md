# NeighborNet — Neighborhood Safety & Complaint App

A simple mobile app for citizens of **Lahore** to report crimes and civic issues, view basic heatmaps, and contribute to community safety.

---

## Overview
**NeighborNet** enables users to:  
- Report street crimes, harassment, theft, and civic problems (e.g., broken streetlights, sanitation issues).  
- Track complaints submitted by themselves or the community.  
- Visualize basic heatmaps to identify areas of concern.  
- Increase community awareness and neighborhood safety.

---

## Tech Stack
- **Frontend:** Flutter, Dart  
- **Backend:** Python, Django, SQLite  
- **Version Control:** Git & GitHub  

---

## Features
- **User Signup & Login**  
- **Submit & View Complaints**  
- **Basic Heatmap Interface (UI)**  
- **Django REST API with SQLite**
